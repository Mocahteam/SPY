L'accessibilité de SPY

Présentation des fonctionnalités d'accessibilité de SPY.

Résumé des fonctionnalités implémentées

Et si l'initiation à la programmation était accessible au plus grand nombre ? Nous travaillons à rendre le plus accessible possible le jeu SPY, il reste encore beaucoup de choses à faire mais nous y travaillons...

Voici la liste des fonctionnalités d'accessibilité actuellement intégrée dans SPY qui favorise son accessibilité :

  • Les interfaces utilisateurs sont stationnaires
  • L'action de glisser-déposer pour ajouter des blocs aux programmes est facultative deux autres méthodes permettant d'ajouter des blocs sont disponibles (voir section Programmation)
  • Toutes les actions clés du jeu sont accessibles au clavier
  • Actions clés accessibles avec des raccourcis claviers (voir Détail des raccourcis claviers)
  • La vitesse d'exécution des programmes est contrôlable avec le mode d'exécution au pas à pas
  • Le jeu est compatible avec les claviers virtuels
  • Le taille des interfaces utilisateurs est réglable
  • La police de caractère est sans sérif
  • Le contraste de couleur entre les textes et les fonds est assuré avec un rapport de contraste compris entre 6.05:1 et 12.77:1
  • Possibilité de relire les consignes à tout moment
  • Module de synthèse vocale pour lire les consignes et les interfaces (voir Détail sur la synthèse vocale)
  • Aucune information essentielle repose uniquement sur la couleur ou le son
  • Bascule possible d'une vue 3D à une vue 2D
  • Compatible avec un lecteur d'écran (testé avec NVDA - voir Détail lecteur d'écran)
  • Exportation de la carte au format textuel (voir Détail lecteur d'écran)
  • Enregistrement automatique de la progression des joueurs

Raccourcis clavier

Généraux :

  • Echap : Sélectionner le menu principal / Annuler action en cours
  • Ctrl+Echap : sortir du canevas du jeu pour revenir dans la page html
  • Home : sélectionner l'élément intéractif le plus prioritaire
  • Touches directionnelles : navigation dans les interfaces du jeu / déplacement des blocs

Contrôle de la caméra :

  • v : Basculer de la vue 3D à la vue 2D
  • z : Avancer la caméra vers l'avant
  • s : Avancer la caméra vers l'arrière
  • q : Avancer la caméra vers la gauche
  • d : Avancer la caméra vers la droite
  • c : Centrer la caméra sur le prochain robot dans la scène
  • a : Pivoter la caméra vers la gauche
  • e : Pivoter la caméra vers la droite
  • r : Zoomer
  • f : Dézoomer

Contrôle de l'exécution :

  • Espace : Lancer/Mettre en pause l'exécution du programme
  • Ctrl + Espace : Exécuter la prochaine instruction du programme
  • Alt + Espace : Accélérer l'exécution du programme
  • Shift + Espace : Stopper l'exécution du programme

Autres :

  • Entrée : Saisir/Relacher un bloc de programmation
  • Suppr : Supprimer un bloc de programmation
  • Retour arrière : Supprimer un bloc de programmation
  • i : Insérer un bloc de programmation
  • h : Afficher les consignes

Synthèse vocale

SPY exploite la synthèse vocale du navigateur web pour vocaliser son contenu. Pour utiliser la synthèse vocale du navigateur vous devez l'activer à l'aide du bouton "Synthèse vocale" situé au dessus du canevas du jeu (pensez à sortir du mode plein écran pour pouvoir accéder à ce bouton).

Une fois la synthèse vocale activée vous pouvez indiquer si vous souhaitez que toute l'interface du jeu soit vocalisée ou seulement les consignes. Vous pouvez également contrôler la vitesse d'élocution de la synthèse vocale.

Une fois la synthèse vocale activée, déplacez vous dans l'interface du jeu à l'aide de la navigation clavier ou de la souris souris pour que les textes soient automatiquement lus. Si vous avez coché la case "Consignes" (par défaut) seuls les textes d'introduction au début des missions seront lus. Si vous avez cohé la case "Tout", n'importe quel élément d'interface (Menus, boutons, listes déroulante, etc.) seront lus par la synthèse vocale.

Pour les utilisateurs utilisant un lecteur d'écran contenant leur propre synthèse vocale, il n'est pas utile d'activer cette option (voir section Détail lecteur d'écran).

Modalités de programmation

Pour résoudre les différentes missions du jeu, les joueurs manipulent un langage de programmation à base de blocs. SPY offre trois méthodes pour manipuler ces blocs et saisir les programmes :

Le Glisser-Déposer "à la souris"

Ce mode d'intéraction permet aux joueurs d'utiliser le pointeur de la souris (ou le doigt pour les dispositifs tactiles) pour glisser les blocs de l'inventaire dans les zones de progammation. Il est également possible avec cette méthode de modifier l'ordre des blocs à l'intérieur d'une zone de programmation pour modifier le programme.

Une fois un premier bloc ajouté dans une zone de programmation, vous pouvez utiliser le double-clic (ou double-tap) sur les blocs de l'inventaire pour ajouter automatiquement ce bloc à la suite du dernier bloc ajouté.

Pour supprimer un bloc, le joueur peut glisser un bloc depuis une zone de programmation à l'extérieur de celle-ci, une alternative à cette méthode et d'éffectuer un clic-droit sur le bloc à supprimer.

Le Glisser-Déposer au clavier

Comme pour le mode "à la souris", cette option permet aux joueurs de se déplacer dans l'interface à l'aide des flèches directionnelles du clavier afin de se positionner sur le bloc à déplacer. Une fois positionner sur un bloc de programmation, appuyer sur la touche "Entrée" attrape le bloc. Utilisez les touches directionnelles du clavier pour déplacer le bloc dans l'interface puis appuyer à nouveau sur "Entrée" pour relacher le bloc. Si le bloc est relaché dans une zone de programmation valide il sera ajouté au programme, sinon il sera supprimé.

L'insertion au clavier

Contrairement aux deux modes précédents, ici le joueur doit d'abord se positionner dans une zone de programmation à l'endroit où il souhaite insérer un nouveau bloc. Il appuit alors sur la touche "i" du clavier pour "insérer" un nouveau bloc. Automatiquement la focus est positionné sur le premier bloc disponible dans l'inventaire, le joueur peut alors parcourir les blocs disponibles et valider son choix avec la touche "Entrée". Le bloc sélectionné sera automatiquement ajouté au programme à l'endroit où la touche "i" avait été pressée.

Cette méthode permet également de modifier l'ordre des blocs dans un programme. Le joueur doit se positionner à l'endroit où il souhaite déplacer un bloc, appuyer sur "i", se positionner sur le bloc à déplacer dans le programme et appuyer sur "Entrée". Le bloc sélectionné sera automatiquement déplacer à l'endroit où la touche "i" avait été pressée.

A noter que si la touche "i" est pressée sur un bloc déjà intégré dans une zone de programmation, le nouveau bloc insérer sera ajouté juste avant le bloc sélectionné.

Lecteur d'écran

Identifier la zone du jeu dans la page Web

Pour accéder au canevas du jeu, naviguez dans la page jusqu'à arriver à l'objet "Application" ou "Zone intéractive du jeu". Appuyez sur "Entrée" pour entrer dans le jeu. Une fois dans le jeu vous pouvez à tout moment appuyer sur : "Shift + Entrée" pour sortir du canevas du jeu et revenir dans le contexte de la page web ; Appuyer sur la touche "Home" pour automatiquement sélectionner l'élément intéractif le plus prioritaire.

Un peu de technique : Lorsqu'un objet du jeu obtient le focus, sa description textuelle est automatiquement envoyé vers la balise html "ScreenReaderSpan" avec le role "status". Cette balise est définie avec les propriétés aria-live="assertive" et aria-atomic=true afin que le lecteur d'écran puisse être informé immédiatement des changements de son contenu et les rendre accessibles au joueur via la synthèse vocale ou une plage braille (testé avec une Brailliant Bi 40X et NVDA).

Description de la scène

L'objectif du jeu est de déplacer un robot dans un labyrinthe, il est donc nécessaire de pouvoir accéder à une description de la scène de jeu et de suivre son évolution au cours de la simulation. Une description textuelle de la scène est en permanance synchronisée avec la balise html "DescMapSpan". Pour consulter l'état de la scène à un instant t, sortir du canevas du jeu avec "Shift + Entrée", le focus devrait automatiquement être positionné sur l'élément "Continuer la navigation" dans la page web. La description textuelle de la carte se trouve juste en dessous. Pour revenir dans le jeu, remonter jusqu'à retrouver la "Zone intéractive du jeu" et appuyé sur "Entrée".
